Diablo Immortal Calculator

An all in one tool to guide players on their progression path and know what costs to expect across various systems.

Currently the tool handles gear ranks, legendary gems (including internal), normal gems and iben fahd's vessels.

Suggestions are welcome for new ideas to add but should ideally be cement progression systems.

If you encounter a bug, create an issue with as much detail as you can.

How to use:
1. Download the project
2. Install Python: https://www.python.org/downloads/
3. Download UPX: (https://github.com/upx/upx/releases)
4. Extract UPX and place folder in the path where the project is located
5. a) Windows: edit the 'compile.bat' script if your UPX version is different to the one pre assigned then run
   b) MAC: edit the 'compile.command' script if your UPX version/hardware is different to the one pre assigned then run
   * MAC users may need to open the Terminal app first and type: 'chmod +x' then drag the .command file onto the terminal and press enter *
6. Locate the compiled executable file in the 'dist' folder
